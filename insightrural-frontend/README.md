# InsightRural Frontend Implementation

A fully modular, standalone web interface for the InsightRural AI-powered educational guidance system for rural students.

## Project Overview

This frontend implements a ChatGPT-style conversational interface where rural students can ask questions related to:
- 🏛️ Colleges & Courses
- 💰 Scholarships
- 🏦 Education Loans
- 🏠 Hostels & Accommodation

The interface is built using pure HTML, CSS, and JavaScript with no frameworks or dependencies, making it accessible in low-resource environments.

## Key Features

1. **Modular Architecture**: Each guidance area is implemented as an independent module
2. **Conversational Interface**: ChatGPT-style chat for intuitive interaction
3. **Student Profile**: Personalization based on student background
4. **Mock Data System**: Simulates AI responses without backend dependencies
5. **Responsive Design**: Works on desktop and mobile devices
6. **Local Storage**: Saves chat history and profile data locally

## How to Run

1. Simply double-click on `index.html` to open in any modern web browser
2. No server setup or installation required
3. Works offline after initial load
